# Google Keyword Extractor -  لاراول 🚀

یک ابزار حرفه‌ای و سئو محور برای استخراج لایه‌ای کلمات کلیدی پیشنهادی گوگل (Google Autocomplete) که با فریم‌ورک محبوب لاراول پیاده‌سازی شده است.

## 🌟 ویژگی‌ها

- **هسته لاراول:** استفاده از معماری MVC برای توسعه و شخصی‌سازی آسان.
- **استخراج لایه‌ای:** کشف کلمات کلیدی دم‌بلند (Long-tail) تا ۱۰ لایه عمق.
- **ترجمه همزمان:** ترجمه زنده نتایج به فارسی برای کوئری‌های انگلیسی و خارجی.
- **UI مدرن:** رابط کاربری جذاب با Tailwind CSS و انیمیشن‌های نرم.
- **خروجی متنوع:** قابلیت کپی گروهی و دانلود نتایج در قالب فایل متن.

## 🚀 راهنمای نصب و اجرا

برای اجرای این پروژه در سیستم خود، مراحل زیر را دنبال کنید:

### ۱. نیازمندی‌ها
- PHP >= 8.2
- Composer
- Node.js & NPM (اختیاری برای توسعه فرانت)

### ۲. نصب وابستگی‌ها
ابتدا وابستگی‌های PHP را نصب کنید:
```bash
composer install
```

### ۳. تنظیمات فایل .env
فایل نمونه را کپی کنید:
```bash
cp .env.example .env
```
سپس کلید اپلیکیشن را تولید کنید:
```bash
php artisan key:generate
```

### ۴. اجرای پروژه
سرور داخلی لاراول را اجرا کنید:
```bash
php artisan serve
```
حالا می‌توانید پروژه را در مرورگر خود در آدرس `http://localhost:8000` مشاهده کنید.

### ۵. دسترسی‌های فایل (در صورت نیاز)
اگر با خطای دسترسی به پوشه storage مواجه شدید، دستور زیر را در لینوکس یا مک اجرا کنید:
```bash
chmod -R 775 storage bootstrap/cache
```

## 🛠 ساختار پروژه
- **Controller:** `app/Http/Controllers/KeywordController.php`
- **View:** `resources/views/google_suggestion_tool.blade.php`
- **Routes:** `routes/web.php`
